Bedrock Storyteller

This is a simple web application that uses AI to generate bedtime stories. It's developed for educational purposes to demonstrate the power of AI in generating human-like text. The solution consists of the following:

  • Advanced NLP Model: Utilizing the Bedrock model amazon.titan-text-lite-v1 for enhanced natural language processing capabilities.
  • Scalable Backend Processing: Implementing a containerized Lambda function using aws-lambda-web-adapter for efficient and flexible backend operations, or using Flask, a Python web framework, for backend development.
  • Secure User Management: Leveraging AWS Cognito for robust authentication, authorization, and user lifecycle management, including login and logout functionalities.
  • Pre-Signup Confirmation: Integrating an AWS Cognito pre-signup Lambda function to ensure seamless user confirmation processes.
  • Efficient Deployment: Employing AWS SAM (Serverless Application Model) for streamlined deployment of the solution, ensuring rapid iterations and updates.
  • Intuitive Frontend Design: Building the frontend with HTML to deliver a user-friendly and engaging interface for seamless interactions.

These components come together to create a powerful and versatile solution that showcases the synergy of advanced AI capabilities, secure user management, efficient deployment practices, and intuitive frontend design. Enjoy!!
